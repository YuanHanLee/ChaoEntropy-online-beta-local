Overview

The program ChaoMI ( Mutual Information proposed by Chao et al.) is written in the R language and the interactive web application is built by using Shiny. The user provides a matrix of two-dimension data, in which cells with positive integers indicate the frequency of interaction between a pair of species (variables). ChaoMI computes the Mutual Information estimators, bootstrap standard error and confidence intervals.

How to Cite

If you use ChaoEntropy Online to obtain results for publication, you should cite the papers (Chao, A., Wang, Y. T. and Jost, L. (2013) Entropy and the species accumulation curve: a novel entropy estimator via discovery rates of new species. Methods in Ecology and Evolution, 4, 1091-1100.) along with the following reference for ChaoEntropy Online:

Chao, A., Lee, Y.-H. and Tseng, K.-S. (2014). ChaoEntropy Online.

To help refine ChaoEntropy Online, your comments or feedbacks would be welcome (please send them to chao@stat.nthu.edu.tw).

Data Settings

Dataset

Some demonstration datasets are used for illustration.

  • Schleuning et al. (2010): This describes a plant-frugivore network from a Kenyan rain forest. Interaction frequency in the quantitative interaction matrix was defined as the number of fruit-eating individuals on a plant species independent of fruit handling (including swallowing, pecking, and dropping fruits). For more detail, you can see the source of this data : http://www.nceas.ucsb.edu/interactionweb/html/schleuning-et-al-2010.html

  • Bluthgen et al. (2004): This describes de ant-plant interaction network in the rainforest at the Australian Canopy Crane in Cape Tribulation, Far North Queensland, Australia (16°07' S, 145°27' E, 80 m a.s.l.). Frequency of interaction was estimated as the number of ant individuals of a particular species collected on a plant species. Data are presented as an interaction frequency matrix, in which cells with positive integers indicate the frequency of interaction between a pair of species, and cells with zeros indicate no interaction. For more detail, you can see the source of this data : http://www.nceas.ucsb.edu/interactionweb/html/bluthgen_2004.html

Demo Data

ChaoMI provides three demo dataset from Schleuning et al. (2010). We according to forest strata deviding the data into three class:

  1. canopy
  2. midstory
  3. understory

You can compare its interaction network from different forest strata.

Upload Data

ChaoMI also provides a upload function. Click the Upload data button, there will show the file choose button to upload the data. User's data MUST be .csv. After upload the data, the file name which your uploaded will be listed in the box.

We provides three dataset to download from the Bluthgen et al. (2004), according to the subfamily of ants deviding the data into three group:

  1. Dolichoderinae(Click to download)
  2. Formicinae(Click to download)
  3. Myrmicinae(Click to download)

Notice: In the upload files, the variable names are better as simple as possible. It can't include symbol like ' ', " ". If the Data Viewer was blank after click Run button, check the variable names and see the demo download files above.

General settings

Method

This is a checkbox for select the method which constructed to estimate mutual information.

  • Chao estimator, for detail see reference 1.
  • ChaoShen estimator, for detail see reference 3.
  • Jackknife estimator, for detail see reference 5.
  • Bias Correct 1 estimator.
  • Bias Correct 2 estimator.
  • Observed estimator.

Bootstraps

Number of bootstraps (say B) is an integer specifying the number of replications for bootstrap resampling scheme in computing variance. Refer to Chao et al. (2013) for details. Default is 100. To save running time, we recommend that 100 or 200 bootstraps will be sufficient for most applications.

Confidence level

The confidence level is a positive number is less than or equal to 1. The default is 0.95.

Output

Data Summary

This tab panel shows basic data information for the selected data. The output variables are interpreted at the first column.

Estimation

This tab panel shows the main output for ChaoMI. It show a table of various mutual information estimators, their standard error and confidence interval which the method you choose. You also can click Download as csv file to download the output table.

Visualization

This tab panel shows the interactive estimator and confidence interval plot.
